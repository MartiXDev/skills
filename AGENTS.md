## Agent skills

### Issue tracker

Issues are tracked in GitHub Issues for this repository, and external pull requests are **not** a triage request surface. See `docs/agents/issue-tracker.md`.

### Triage labels

Triage uses the default five-label vocabulary (`needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`). See `docs/agents/triage-labels.md`.

### Domain docs

This repository uses a multi-context domain-doc layout via `CONTEXT-MAP.md` with per-context `CONTEXT.md` and ADR locations. See `docs/agents/domain.md`.
