## MartiX .NET 10 and C# 14 skill

This skill provides standalone-first .NET 10+ and C# 14+ guidance for:

- Reviewing and modernizing SDK-style .NET projects.
- Shaping ASP.NET Core or Minimal API services.
- Refactoring async code, cancellation flows, or background work.
- Tuning hot paths, spans, collections, or allocation-sensitive code.
- Reviewing EF Core, JSON contracts, diagnostics, testing, and security defaults.

## When to use this skill

- Working on .NET 10+ or C# 14+ projects.
- Modernizing existing .NET code to current best practices.
- Scaffolding new ASP.NET Core services or libraries.
- Reviewing options pattern, TypedResults, `ValidateOnStart()`, HTTP resilience,
  `WebApplicationFactory`, extension blocks, channels, or problem details.
