## MartiX FluentValidation skill

This skill provides standalone-first FluentValidation guidance for:

- Authoring validators using the fluent rule API.
- Selecting built-in validators and understanding their behavior.
- Defining RuleSets and controlling cascade modes.
- Integrating FluentValidation with ASP.NET Core model binding.
- Writing async validators and managing execution order.
- Extending FluentValidation with custom validators and extensions.
- Localizing validation messages.
- Unit testing validators effectively.
- Planning upgrades between FluentValidation major versions.

## When to use this skill

- Creating or reviewing FluentValidation validators.
- Debugging validation failures or unexpected cascade behavior.
- Integrating FluentValidation into new or existing ASP.NET Core projects.
- Writing tests for validation logic.
