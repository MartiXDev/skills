## MartiX SharePoint Server skill

This skill provides standalone-first guidance for SharePoint Server on-premises:

- Authoring farm solutions and WSP packages.
- Implementing Feature Framework elements (features, elements.xml, site columns,
  content types, list definitions).
- Writing event receivers and timer jobs.
- Applying master pages, page layouts, and branding.
- Deploying site artifacts and managed metadata.
- Migrating toward SharePoint Online or modern SharePoint patterns.

## When to use this skill

- Developing or reviewing SharePoint Server 2019 / Subscription Edition solutions.
- Packaging and deploying WSP farm solutions.
- Implementing Feature Framework for site provisioning.
- Debugging event receivers or timer jobs on SharePoint Server.
