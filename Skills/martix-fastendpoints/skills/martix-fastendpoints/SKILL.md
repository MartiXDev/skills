## MartiX FastEndpoints skill

This skill provides standalone-first FastEndpoints guidance for:

- Bootstrapping and configuring FastEndpoints applications.
- Authoring endpoint contracts, validators, mappers, and processors.
- Choosing transport layers (REST, gRPC, messaging).
- Securing endpoints with permissions, policies, and JWT.
- Testing with the FastEndpoints testing utilities.
- Versioning APIs and managing OpenAPI documentation.
- Targeting Native AOT scenarios.

## When to use this skill

- Building or reviewing FastEndpoints-based ASP.NET Core APIs.
- Designing endpoint request/response contracts.
- Adding validators, pre/post processors, or global processors.
- Migrating from controller-based APIs to FastEndpoints.
