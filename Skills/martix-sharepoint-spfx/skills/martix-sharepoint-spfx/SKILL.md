## MartiX SharePoint Framework (SPFx) skill

This skill provides standalone-first guidance for SPFx development:

- Scaffolding SPFx web parts, application customizers, and field customizers.
- Building component UIs with React and TypeScript.
- Integrating with Microsoft Teams and Viva Connections.
- Packaging and deploying to SharePoint App Catalog.
- Applying theming and Fluent UI design tokens.
- Accessing SharePoint data via Microsoft Graph and PnPjs.
- Migrating from classic web parts or AngularJS to SPFx.

## When to use this skill

- Building or reviewing SPFx web parts or extensions.
- Integrating SPFx components into Microsoft Teams or Viva Connections.
- Deploying to SharePoint Online tenant or site app catalog.
- Applying SharePoint theming or Fluent UI components.
