## MartiX SharePoint PnP skill

This skill provides standalone-first guidance for the PnP toolchain:

- Scripting SharePoint Online and On-Premises operations with PnP PowerShell.
- Automating Microsoft 365 workloads with CLI for Microsoft 365 (`m365`).
- Accessing SharePoint and Graph APIs from JavaScript/TypeScript using PnPjs.
- Creating and applying PnP provisioning templates.
- Authenticating with SharePoint via certificate, device code, or managed identity.
- Choosing between PnP PowerShell, CLI for M365, and PnPjs for a given task.

## When to use this skill

- Writing PnP PowerShell scripts for SharePoint automation.
- Using CLI for Microsoft 365 in pipelines or terminal workflows.
- Building JavaScript solutions that access SharePoint with PnPjs.
- Creating or applying PnP provisioning templates.
- Selecting the right PnP tool for a provisioning or management task.
