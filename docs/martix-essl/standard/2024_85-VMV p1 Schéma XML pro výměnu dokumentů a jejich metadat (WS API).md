**Schéma XML pro výměnu dokumentů a jejich metadat (WS API)**

## Schéma je určeno pro výměnu dokumentů a jejich metadat mezi informačními systémy spravujícími dokumenty a eSSL, přičemž se použije také pro export, import a přenos dokumentů. V tom případě se využijí pouze datové prvky definované ve schématu.
