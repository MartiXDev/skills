# Archive

This folder contains historical documents that are no longer active guidance but
are retained for context and understanding previous decisions.

Archive is a location, not a document role. Historical documents belong here;
active documents belong in the main `docs/` tree.

## Access guidance

When referencing archived documents:

- Link with context: "See the [archived coordination plan](./archive/plans/skill-portfolio-coordination-plan.md) for historical context."
- Do not present archived content as current guidance
- Do not copy archived content into active documents without updating it first
- Preserve archived documents as immutable records of past decisions
