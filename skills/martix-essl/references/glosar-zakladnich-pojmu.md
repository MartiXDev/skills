# Glosář základních pojmů

| Pojem | Stručný význam |
| --- | --- |
| Dokument | Zaznamenaná informace vytvořená nebo doručená původci |
| Komponenta | Souborová nebo jiná část digitální reprezentace dokumentu |
| Spis | Celek, který sdružuje související dokumenty |
| Typový spis | Strukturovaný spis podle šablony a pravidel členění |
| Díl typového spisu | Časově členěná část typového spisu |
| Věcná skupina | Kategorie pro třídění dokumentů a spisů |
| Spisový znak | Označení pro věcné zatřídění |
| Skartační znak | Označení pro způsob vyřazení |
| Skartační lhůta | Doba, po kterou musí být dokument uložen |
| Skartační režim | Kombinace pravidel pro budoucí vyřazení nebo archivaci |
| Jednoznačný identifikátor | Identifikátor, který je spojen s konkrétní entitou nebo dokumentem |
| Hlavička metadat | Soubor metadat společný pro daný typ entity |
| SIP | Submission Information Package pro export, import nebo přenos |
| Transakční protokol | Auditní stopa akcí nad dokumenty a entitami |
| Ztvárnění | Reprezentace obsahu nebo metadat v určité podobě |
| Křížový odkaz | Vazba mezi entitami v rámci eSSL |
| Zajišťovací prvek | Podpis, pečeť, časové razítko nebo jiný prvek ověřující důvěryhodnost |

## Kdy glosář použít

- Než začneš mapovat pojem do databáze nebo API.
- Když máš pocit, že dvě entity splývají.
- Když review naráží na nejasný termín ze standardu.
